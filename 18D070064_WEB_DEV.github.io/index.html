E-Notes

I am very forgetful, so I need to make a lot of notes, all the same, I was also very slow at typing, so handwritten notes were my only tool, but this lockdown has shown me the immense advantage of online over offline. So I have decided to shift to e-note taking for atleast one course the coming sem, in case it commences. Now, how did I prepare? - I used this blog as my basic guide. So first prerequisite was to touch-type at atleast 70-80WPM. I started to learn touch-typing in beginning of June casually and my current speed has improved to 50WPM with >95% accuracy, I am confident it will attain the desired threshold of 70-80WPM within 1 month

Coming to softwares, it was clear that fancy things like evernote and onenote won't work, some coding support is needed. So I configured Vim plugins in WSL as shown by Gilles in his blog. However, being in electrical engineering, I looked for some latex support for circuit diagrams, luckily, I found circuitikz package which can be used to insert circuits in diagram. Next, as engineering students, we also need to make lots of parametric plots, with real data for this, I have decided to use pre-configured MatLab and Wolfram Mathematica files for quality and interactive plots, I know this will increase time, but I will get quality interactive plot compared to static hand-made plot with axes inclined at 45°.

I hope this endeavour is viable
